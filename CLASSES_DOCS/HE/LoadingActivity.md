# תיאור מחלקה: LoadingActivity

## 1. מידע כללי
*   **שם המחלקה:** `LoadingActivity`
*   **סוג:** Activity (מסך המתנה)
*   **תפקיד באפליקציה:** מסך זה מופיע בזמן שהרדיו נטען, כדי שהמשתמש ידע שהאפליקציה עובדת ולא נתקעה.

## 2. הסבר במילים פשוטות
דמיינו ש-`LoadingActivity` היא **חדר המתנה**. כשאתם מבקשים לשמוע רדיו, לוקח זמן לאות להגיע. כדי שלא תסתכלו על מסך ריק, מראים לכם את המסך הזה עד שהמוזיקה מוכנה.
