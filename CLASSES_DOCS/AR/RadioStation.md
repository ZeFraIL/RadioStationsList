# وصف الفئة: RadioStation

## 1. معلومات عامة
*   **اسم الفئة:** `RadioStation`
*   **النوع:** Normal Class (نموذج بيانات)
*   **الغرض من الفئة:** تعمل هذه الفئة كـ "صندوق" لتخزين المعلومات حول محطة راديو معينة (الاسم، رابط البث، إلخ).

## 2. المتغيرات
*   **`name`**: اسم المحطة.
*   **`streamUrl`**: الرابط المباشر لتشغيل الموسيقى.
*   **`homepage`**: رابط الموقع الإلكتروني للمحطة.

## 3. شرح بكلمات بسيطة
تخيل أن `RadioStation` هي **بطاقة عمل**. مكتوب على البطاقة كل التفاصيل المهمة للمحطة. يستخدم التطبيق هذه البطاقة لمعرفة الاسم الذي يجب عرضه والرابط الذي يجب تشغيله.
