# Описание класса: GenreActivity

## 1. Общая информация
*   **Имя класса:** `GenreActivity`
*   **Тип:** Activity (Начальный экран)
*   **Назначение:** Это "входная дверь" приложения. Задача этого экрана — позволить пользователю выбрать жанр музыки перед тем, как загрузить основной список станций.
*   **Взаимодействие с другими компонентами:**
    *   Запускает `MainActivity`.
    *   Передает выбранный жанр (текстовую строку) через `Intent`.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `rbAll`, `rbJazz`, `rbNews` | `RadioButton` | Кнопки выбора (переключатели) для жанров. | `initComponents()` |
| `bReady` | `Button` | Кнопка "Start" для подтверждения выбора. | `onCreate()` |
| `genre` | `String` | Хранит название выбранного жанра ("all", "jazz" или "news"). | Слушатель нажатия |

## 3. Методы класса
### Имя метода: `onCreate`
*   **Тип:** `protected`
*   **Что делает метод:** Устанавливает макет экрана, инициализирует кнопки и описывает логику перехода.
*   **Логика:** При нажатии на кнопку проверяется, какой из переключателей (`rbJazz`, `rbNews`) активен. Затем создается "письмо" (`Intent`), в которое вкладывается название жанра, и отправляется в `MainActivity`.

## 4. Общая логика
1. Приложение запускается.
2. Пользователь видит три варианта: Все, Джаз или Новости.
3. Пользователь выбирает один и нажимает "Start".
4. Приложение "запоминает" выбор и открывает главный экран со списком.

## 5. Объяснение простыми словами
Представьте, что `GenreActivity` — это **администратор** на входе в большой концертный зал. Прежде чем пустить вас внутрь (в `MainActivity`), администратор спрашивает: "Какую музыку вы предпочитаете?". Вы отвечаете, он дает вам номерок и пропускает дальше.
