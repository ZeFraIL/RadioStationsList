# תיאור מחלקה: RadioStation

## 1. מידע כללי
*   **שם המחלקה:** `RadioStation`
*   **סוג:** Normal Class (מודל נתונים)
*   **תפקיד באפליקציה:** מחלקה זו משמשת כ"קופסה" לאחסון מידע על תחנת רדיו ספציפית (שם, קישור לשידור וכו').

## 2. משתנים
*   **`name`**: שם התחנה.
*   **`streamUrl`**: הכתובת הישירה להשמעת המוזיקה.
*   **`homepage`**: כתובת האתר של התחנה.

## 3. הסבר במילים פשוטות
דמיינו ש-`RadioStation` היא **כרטיס ביקור**. על הכרטיס רשומים כל הפרטים החשובים של התחנה. האפליקציה משתמשת בכרטיס הזה כדי לדעת איזה שם להציג ואיזה קישור לנגן.
